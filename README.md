# SecureWeb - Prototype

Load this folder in Chrome (chrome://extensions → Developer mode → Load unpacked).

## Purpose
Prototype browser extension to:
- Detect suspicious JS and DOM injections
- Block auto-downloads
- Enforce privacy toggles (camera/mic/location)
- Provide cookie/session vault helpers using Web Crypto
- Profile typing/mouse dynamics (local-only) for anomaly detection
- Role-based access checks and logging (logcat)

## Notes
- This is a prototype for testing and learning.
- Test in a controlled environment.
